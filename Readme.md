### Build executables

- make

### Remove executables

- make clean

### Update PATH to point to your custom shell counterpart

- export PATH=$(pwd):$PATH

### Utilities supported

#### cat

### Example Usage

- cat zen.txt
- cat -o zen1.txt zen.txt
- cat -a zen1.txt zen.txt
- cat -o zen1.txt zen.txt test.txt
- cat -a zen1.txt zen.txt test.txt
